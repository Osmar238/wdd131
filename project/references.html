Documentación del Proyecto

Este sitio web fue diseñado y desarrollado por Osmar Arevalo Molina como proyecto final para la asignatura WDD 131: Dynamic Web Fundamentals (BYU-Pathway Worldwide).

1. Recursos Gráficos y Multimedia

  • Fuente de Imágenes: Todas las fotografías utilizadas en este proyecto (portada, productos y panadería) fueron recopiladas a través del motor de búsqueda de Google Imágenes.
  • Descargo de Responsabilidad (Disclaimer): Estas imágenes se utilizan estrictamente con fines educativos y demostrativos bajo el principio de uso justo (Fair Use). No se pretende infringir derechos de autor ni obtener lucro comercial con este proyecto escolar.
  • Iconografía: Se utilizaron gráficos vectoriales (SVG) de código abierto para los iconos de éxito y navegación.

2. Tecnologías Implementadas

  • HTML5 Semántico: Estructura del sitio optimizada para accesibilidad y SEO (etiquetas main, nav, header, footer).
  • CSS3 Moderno:
    • Uso de CSS Grid para la galería responsiva de productos.
    • Uso de Flexbox para la navegación y alineación de elementos.
    • Variables CSS (:root) para la gestión consistente de la paleta de colores.
  • JavaScript (ES6+):
    • Manipulación del DOM para la inyección dinámica de tarjetas de productos.
    • Uso de localStorage para la persistencia de datos del carrito de compras.
    • Funciones asíncronas simuladas (setTimeout) para mejorar la experiencia de usuario (UX).

3. Herramientas de Desarrollo

  • Editor de Código: Visual Studio Code.
  • Control de Versiones: Git & GitHub.
  • Fuentes Tipográficas: Playfair Display y Segoe UI (vía Google Fonts).
